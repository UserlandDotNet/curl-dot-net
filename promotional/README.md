# Promotional Content

This folder contains long-form articles, blog posts, and social content that help launch CurlDotNet and the broader UserlandDotNet vision.

## Structure

- `BLOG_POSTS/` – Platform-specific tutorials (Dev.to, HackerNoon) for C#, F#, VB.NET.
- `BLOG_POST_INTRO.md` – Personal blog / GitHub intro post.
- `PROJECT_ARTICLE.md` – Long-form story suitable for IronSoftware.com.
- `VISIBILITY_GUIDE.md` – Checklist of communities, newsletters, and portals to contact.
- `GITHUB_GIST.md` – Concise snippet for sharing on GitHub Gists or social media.

## Usage

1. Customize wording with current release details (version, features, roadmap).
2. Cross-link the repo (`https://github.com/jacob-mellor/curl-dot-net`) and the new `https://github.com/UserlandDotNet` organization.
3. Reference Jacob Mellor’s bio pages (IronSoftware, LinkedIn, GitHub) for PR consistency.
4. Link back to the latest manual guides for deeper dives:
   - [Curl String Deep Dive](../manual/06-Curl-String-Deep-Dive.md) when articles mention copy/paste workflows.
   - [Fluent Builder Cookbook](../manual/07-Fluent-Builder-Cookbook.md) for IntelliSense-heavy tutorials.
   - [LibCurl Playbook](../manual/05-LibCurl-Playbook.md) when showcasing reusable service clients.
   - [Parity Validation Playbook](../manual/08-Parity-Validation-Playbook.md) for trust-focused announcements.

Feel free to add more subfolders (e.g., `/press`, `/videos`) as the marketing plan expands.
